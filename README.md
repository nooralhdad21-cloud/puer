# 🏥 نظام مطابقة فواتير مكتب بيور العلمي (Pure Match)

نظام ويب متطور ومخصص لموظفي **مكتب بيور العلمي للأدوية**، يهدف إلى تسهيل عملية مطابقة الفواتير الصادرة من المكتب مع الفواتير الواصلة فعلياً إلى المذاخر والصيدليات في مختلف المحافظات.

## 🚀 المميزات الرئيسية
* **معالجة الكشوفات الضخمة:** رفع ملفات Excel الخاصة بالمكتب واستخراج البيانات تلقائياً.
* **واجهة ميدانية (Mobile-First):** تصميم متوافق تماماً مع الهواتف الذكية لتسهيل الاستخدام أثناء الجولات الميدانية.
* **نظام هوية الموظف:** توثيق كل عملية مطابقة باسم الموظف المسؤول لضمان الدقة والمسؤولية.
* **تقارير احترافية:** سحب نتائج المطابقة بصيغة **Excel** للحسابات و **PDF** رسمي للإدارة بضغطة زر.
* **ملاحظات حية:** إمكانية إضافة ملاحظات عن حالة الشحنة (نقص، تضرر، تأخير) فوراً عند الاستلام.

## 🛠 التكنولوجيا المستخدمة
* **Frontend:** HTML5, Tailwind CSS (للتصميم الاحترافي).
* **Logic:** JavaScript (Vanilla JS).
* **Libraries:** * `SheetJS` لمعالجة ملفات الأكسل.
    * `jsPDF` لتوليد تقارير PDF احترافية.
